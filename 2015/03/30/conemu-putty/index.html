ConEmu和Putty

By Daniel Cai

发表于 3月 30 2015

文章目录

  3. 3. 配置ConEmu

最近一直在Windows平台下访问AWS的ec2服务器。putty作为一款远程连接利器可以帮助我们很方便地连接ec2服务器。但是如果有很多台服务器需要管理，而且每次都需要key文件进行远程连接，putty就显得不是很方便了。我们今天要介绍一款同样免费的软件，它可以配合putty一起使用，这样就可以在Windows平台下同时管理很多台服务器。

软件叫ConEmu。下面我们就来演示怎么利用ConEmu和putty配合进行远程主机连接。相信你在Windows平台下一定会喜欢上这样的组合。

下载并安装putty和ConEmu

自行搜索并且安装。

配置putty

按照下图所示，保存你的会话连接，如果需要添加key文件，展开ssh,然后导入你的文件。最后记得点击保存按钮。

配置ConEmu

打开ConEmu软件，找到设置。在startup下面有一个叫做tasks的选项。点击tasks选项，然后你就可以添加相应的task。

你需要配置如下参数
Task Name : Putty(可以自己根据需要填写)
Task Parameters : 不需要填写
Commands : putty.exe -new_console -load “yoursavedsession”

配置好参数，点击保存按钮。在你创建新的连接时就可以直接调用它。

非常赞的组合吧，赶紧尝试一下吧。
